## 1. 소개 및 동기

**스플라인(Spline)**은 원래 물리적 제도 도구로, 몇 개의 제어점에 고정된 길고 유연한 자의 장력이 해당 점들을 통과하는 매끄러운 곡선을 만든다.

수치 해석에서 스플라인은 데이터 점 사이에서 격렬하게 진동할 수 있는 단일 고차 다항식(룽게 현상) 대신, **저차 다항식**을 **구간별(Piecewise)** 방식으로 적합하여 진동을 최소화한다.

> **핵심 통찰:** 단순한 선형 스플라인(데이터 점을 직선 구간으로 연결)은 특히 데이터 범위의 경계 근처에서 단일 고차 다항식보다 **더 나은 근사**를 제공할 수 있다 (그림 18.1).

근본적인 절충:

| 접근법 | 장점 | 단점 |
|---|---|---|
| 단일 고차 다항식 | 전역적으로 매끄러움 | 진동(룽게 현상) |
| 구간별 저차 스플라인 | 진동 없음, 국소적 제어 | 매듭(Knot)에서의 매끄러움을 강제해야 함 |

## 2. 연속성 조건

두 구간별 다항식 구간이 한 점(**매듭, Knot**이라 함)에서 만날 때, 연결이 얼마나 매끄러운지를 지정해야 한다. 이것은 **$C^k$ 연속성**으로 기술된다.

- **$C^0$ 연속성:** 곡선의 위치(함수값)가 매듭에서 연속이다.

$$f_1 = f_2$$

- **$C^1$ 연속성:** 위치 **및** 1차 도함수가 매듭에서 연속이다.

$$f_1 = f_2, \quad f_1' = f_2'$$

- **$C^2$ 연속성:** 위치, 1차 도함수, **및** 2차 도함수가 매듭에서 연속이다.

$$f_1 = f_2, \quad f_1' = f_2', \quad f_1'' = f_2''$$

- **$C^n$ 연속성:** 0차, 1차, 2차, ..., $n$차 도함수가 모두 매듭에서 연속이다.

> **시각적 해석:**
> - $C^0$: 곡선이 연결되어 있지만 뾰족한 모서리(꺾임)가 있을 수 있다.
> - $C^1$: 곡선이 연결되어 있고 접선 방향이 일치하여 눈에 보이는 모서리가 없지만, 곡률이 갑자기 변할 수 있다.
> - $C^2$: 곡선이 연결되어 있고, 접선 방향이 일치하며, 곡률도 연속이어서 시각적으로 매우 매끄럽다.

## 3. 선형 스플라인(Linear Spline)

**필요 조건:** $C^0$ 연속성

### 설정

$n$개의 데이터 점을 고려한다: $(x_1, f_1), (x_2, f_2), \dots, (x_n, f_n)$.

- $n - 1$개의 구간이 있다.
- 각 구간 $i$는 자체 스플라인 함수 $S_i(x)$를 가진다.

### 공식

$i$번째 구간 $[x_i,\; x_{i+1}]$에서:

$$S_i(x) = f_i + \frac{f_{i+1} - f_i}{x_{i+1} - x_i}(x - x_i)$$

이것은 단순히 연속된 데이터 점 사이의 **선형 보간**이다.

### 성질

- 자동으로 $C^0$ 연속성을 만족한다(구간들이 끝점을 공유하므로).
- 매듭에서 **매끄럽지 않다** — 두 구간이 만나는 곳에서 1차 도함수가 일반적으로 불연속이다.

> **한계:** 매듭에서의 매끄러움 부족은 매끄러움을 보장하기 위한 더 높은 차수의 다항식 스플라인(이차 및 삼차)의 사용을 동기부여한다.

## 4. 이차 스플라인(Quadratic Spline)

**필요 조건:** $C^1$ 연속성

### 4.1 일반 형태

각 구간 $i$에서 2차 다항식을 사용한다:

$$S_i(x) = a_i + b_i(x - x_i) + c_i(x - x_i)^2$$

각 구간에 **3개의 미지수**($a_i$, $b_i$, $c_i$)가 있다.

$n$개의 데이터 점과 $n-1$개의 구간이 있으므로, 총 미지수는 $3(n-1)$이다.

### 4.2 조건 유도

모든 미지수를 결정하려면 $3(n-1)$개의 방정식이 필요하다.

---

**조건 (i) — 왼쪽 끝점에서의 함수값:**

$$S_i(x_i) = a_i = f_i, \quad \text{for } i = 1, 2, \dots, n-1$$

이것은 $n - 1$개의 방정식을 제공하며, 즉시 $a_i = f_i$를 결정한다.

---

**조건 (ii) — 내부 매듭 $x_{i+1}$에서의 $C^0$ 연속성:**

$$S_i(x_{i+1}) = S_{i+1}(x_{i+1}) = f_{i+1}$$

다항식 형태를 대입하고 $h_i := x_{i+1} - x_i$로 놓으면:

$$f_i + b_i h_i + c_i h_i^2 = f_{i+1}, \quad \text{for } i = 1, 2, \dots, n-1$$

이것은 $n - 1$개의 방정식을 제공한다.

---

**조건 (iii) — $(n-2)$개의 내부 매듭에서의 $C^1$ 연속성:**

각 구간의 도함수는:

$$S_i'(x) = b_i + 2c_i(x - x_i)$$

각 내부 매듭 $x_{i+1}$에서 인접 구간의 도함수를 같게 놓으면:

$$S_i'(x_{i+1}) = S_{i+1}'(x_{i+1})$$

$$b_i + 2c_i h_i = b_{i+1}, \quad \text{for } i = 1, 2, \dots, n-2$$

이것은 $n - 2$개의 방정식을 제공한다.

---

**방정식 수:**

| 출처 | 수 |
|---|---|
| (i) 왼쪽 끝점 값 | $n - 1$ |
| (ii) $C^0$ 연속성 | $n - 1$ |
| (iii) $C^1$ 연속성 | $n - 2$ |
| **합계** | $3(n-1) - 1$ |

방정식이 **1개 부족**하다. 따라서 추가 조건이 필요하다.

---

**조건 (iv) — 추가 조건:**

$c_1 = 0$을 선택한다. 이는 첫 번째 구간이 **선형 근사**가 됨을 의미한다:

$$S_1(x) = f_1 + b_1(x - x_1)$$

이것으로 시스템이 닫히고 모든 계수를 순차적으로 계산할 수 있다.

### 4.3 예제 18.2 — 이차 스플라인

**주어진 조건:** 4개의 데이터 점 $(x_1, f_1),\, (x_2, f_2),\, (x_3, f_3),\, (x_4, f_4)$

- 3개의 구간
- $3 \times 3 = 9$개의 미지수 (단, $a_i = f_i$로 6개로 축소)

**세 스플라인 함수 ($a_i = f_i$ 적용 후):**

$$S_1(x) = f_1 + b_1(x - x_1) + c_1(x - x_1)^2$$

$$S_2(x) = f_2 + b_2(x - x_2) + c_2(x - x_2)^2$$

$$S_3(x) = f_3 + b_3(x - x_3) + c_3(x - x_3)^2$$

**$C^0$ 연속성 (조건 ii)으로부터:**

$$f_2 - f_1 = b_1 h_1 + c_1 h_1^2$$

$$f_3 - f_2 = b_2 h_2 + c_2 h_2^2$$

$$f_4 - f_3 = b_3 h_3 + c_3 h_3^2$$

**$C^1$ 연속성 (조건 iii)으로부터:**

$$b_1 + 2c_1 h_1 - b_2 = 0$$

$$b_2 + 2c_2 h_2 - b_3 = 0$$

**추가 조건 (조건 iv)으로부터:**

$$c_1 = 0$$

이것은 6개의 미지수($b_1, c_1, b_2, c_2, b_3, c_3$)에 대한 **6개의 방정식**을 제공한다.

**행렬 / 벡터 형태:**

$$\begin{pmatrix} f_2 - f_1 \\ f_3 - f_2 \\ f_4 - f_3 \\ 0 \\ 0 \\ 0 \end{pmatrix} = \begin{pmatrix} h_1 & h_1^2 & 0 & 0 & 0 & 0 \\ 0 & 0 & h_2 & h_2^2 & 0 & 0 \\ 0 & 0 & 0 & 0 & h_3 & h_3^2 \\ 1 & 2h_1 & -1 & 0 & 0 & 0 \\ 0 & 0 & 1 & 2h_2 & -1 & 0 \\ 0 & 1 & 0 & 0 & 0 & 0 \end{pmatrix} \begin{pmatrix} b_1 \\ c_1 \\ b_2 \\ c_2 \\ b_3 \\ c_3 \end{pmatrix}$$

> **참고:** 이것은 표준적인 기법(가우스 소거법, LU 분해 등)을 사용하여 풀 수 있는 선형 시스템 $\mathbf{A}\mathbf{x} = \mathbf{b}$이다.

## 5. 삼차 스플라인(Cubic Spline)

**필요 조건:** $C^2$ 연속성

### 5.1 일반 형태

각 구간 $i$에서 3차 다항식을 사용한다:

$$S_i(x) = a_i + b_i(x - x_i) + c_i(x - x_i)^2 + d_i(x - x_i)^3$$

각 구간에 **4개의 미지수**($a_i$, $b_i$, $c_i$, $d_i$)가 있다.

$n$개의 데이터 점과 $n-1$개의 구간이 있으므로, 총 미지수는 $4(n-1)$이다.

삼차 스플라인의 핵심 성질:

- 삼차 곡선은 **컴퓨터 그래픽스에서 널리 사용**된다.
- 구간별 삼차 다항식은 **$C^2$ 연속성**을 허용한다.
- 계산 비용과 매끄러움 사이의 좋은 **절충**이다.

### 5.2 조건 유도 (i -- iv)

$h_i := x_{i+1} - x_i$로 놓는다.

---

**조건 (i) — 왼쪽 끝점에서의 함수값:**

$$S_i(x_i) = a_i = f_i, \quad \text{for } i = 1, 2, \dots, n-1$$

$n - 1$개의 방정식을 제공한다. 즉시: $a_i = f_i$.

---

**조건 (ii) — 매듭에서의 $C^0$ 연속성:**

$$S_i(x_{i+1}) = f_{i+1}$$

$$f_i + b_i h_i + c_i h_i^2 + d_i h_i^3 = f_{i+1}, \quad \text{for } i = 1, 2, \dots, n-1 \quad \cdots (**)$$

$n - 1$개의 방정식을 제공한다.

---

**조건 (iii) — 내부 매듭에서의 $C^1$ 연속성:**

$$S_i'(x) = b_i + 2c_i(x - x_i) + 3d_i(x - x_i)^2$$

$x_{i+1}$에서:

$$S_i'(x_{i+1}) = b_i + 2c_i h_i + 3d_i h_i^2 = S_{i+1}'(x_{i+1}) = b_{i+1}$$

$$\therefore \quad b_i + 2c_i h_i + 3d_i h_i^2 = b_{i+1}, \quad \text{for } i = 1, 2, \dots, n-2 \quad \cdots (*)$$

$n - 2$개의 방정식을 제공한다.

---

**조건 (iv) — 내부 매듭에서의 $C^2$ 연속성:**

$$S_i''(x) = 2c_i + 6d_i(x - x_i)$$

$x_{i+1}$에서:

$$S_i''(x_{i+1}) = 2c_i + 6d_i h_i = S_{i+1}''(x_{i+1}) = 2c_{i+1}$$

$$\therefore \quad c_{i+1} = c_i + 3d_i h_i \quad \cdots (***)$$

$$\Longrightarrow \quad d_i = \frac{c_{i+1} - c_i}{3 h_i}$$

$n - 2$개의 방정식을 제공한다.

---

**현재까지의 방정식 수:**

| 출처 | 수 |
|---|---|
| (i) $a_i = f_i$ | $n - 1$ |
| (ii) $C^0$ 연속성 | $n - 1$ |
| (iii) $C^1$ 연속성 | $n - 2$ |
| (iv) $C^2$ 연속성 | $n - 2$ |
| **합계** | $4n - 6$ |

미지수: $4(n-1) = 4n - 4$.

**2개의 방정식이 더** 필요하다 (끝 조건).

### 5.3 미지수 소거 — $c_i$에 대한 삼대각 시스템

핵심 전략은 모든 미지수를 $c_i$만으로 표현한 다음, 삼대각 시스템을 푸는 것이다.

**단계 1: $d_i$를 $c_i$로 표현.**

조건 (iv)로부터:

$$d_i = \frac{c_{i+1} - c_i}{3h_i}$$

**단계 2: $b_i$를 $c_i$로 표현.**

$d_i$를 $C^0$ 연속성 방정식 $(**)$에 대입:

$$f_{i+1} = f_i + b_i h_i + c_i h_i^2 + \frac{c_{i+1} - c_i}{3} h_i^2$$

$$= f_i + b_i h_i + \frac{h_i^2}{3}(c_{i+1} + 2c_i)$$

$b_i$에 대해 풀면:

$$b_i = \frac{f_{i+1} - f_i}{h_i} - \frac{h_i}{3}(2c_i + c_{i+1})$$

**단계 3: 삼대각 방정식 유도.**

$C^1$ 연속성 조건 $(*)$에서 $d_i$를 대입하면:

$$b_{i+1} = b_i + h_i(c_i + c_{i+1})$$

$b_i$ 공식의 인덱스를 1 줄이면:

$$b_{i-1} = \frac{f_i - f_{i-1}}{h_{i-1}} - \frac{h_{i-1}}{3}(2c_{i-1} + c_i)$$

그리고 마찬가지로:

$$b_i = b_{i-1} + h_{i-1}(c_{i-1} + c_i)$$

대수적 조작($b_i$와 $b_{i-1}$의 표현을 대입하고 정리) 후, 결과는:

$$3\frac{f_{i+1} - f_i}{h_i} - 3\frac{f_i - f_{i-1}}{h_{i-1}} = h_{i-1}\,c_{i-1} + 2(h_{i-1} + h_i)\,c_i + h_i\,c_{i+1}$$

$i = 2, 3, \dots, n-2$에 대해.

> **참고:** 좌변은 **분할 차분**을 포함한다:
> $$3\,f[x_{i+1}, x_i] - 3\,f[x_i, x_{i-1}]$$
> 여기서 $f[x_{i+1}, x_i] = \dfrac{f_{i+1} - f_i}{h_i}$.

이것은 미지수 $c_1, c_2, \dots, c_{n-1}$($n - 1$개)에 대한 $n - 3$개의 방정식을 제공한다. 2개의 끝 조건과 합치면 $n - 1$개의 미지수에 대한 $n - 1$개의 방정식을 얻는다.

**행렬 형태 (자연 끝 조건 $c_1 = 0$, $c_n = 0$인 경우):**

$$\begin{pmatrix} 0 \\ 3(f[x_3,x_2] - f[x_2,x_1]) \\ 3(f[x_4,x_3] - f[x_3,x_2]) \\ \vdots \\ 3(f[x_n,x_{n-1}] - f[x_{n-1},x_{n-2}]) \\ 0 \end{pmatrix} = \begin{pmatrix} 1 & & & & \\ h_1 & 2(h_1+h_2) & h_2 & & \\ & h_2 & 2(h_2+h_3) & h_3 & \\ & & \ddots & \ddots & \ddots \\ & & & h_{n-2} & 2(h_{n-2}+h_{n-1}) & h_{n-1} \\ & & & & & 1 \end{pmatrix} \begin{pmatrix} c_1 \\ c_2 \\ c_3 \\ \vdots \\ c_{n-1} \\ c_n \end{pmatrix}$$

> **중요:** 계수 행렬은 **삼대각(Tridiagonal)**이고 **대각 우세(Diagonally Dominant)**이므로, **토머스 알고리즘(Thomas Algorithm)**을 사용하여 $O(n)$ 연산으로 매우 효율적으로 풀 수 있다.

**$c_i$를 풀고 나면, 다른 계수를 복원:**

$$d_i = \frac{c_{i+1} - c_i}{3h_i}, \quad i = 1, 2, \dots, n-1$$

$$b_i = \frac{f_{i+1} - f_i}{h_i} - \frac{h_i}{3}(2c_i + c_{i+1}), \quad i = 1, 2, \dots, n-1$$

### 5.4 끝 조건(End Conditions, 18.4.2절)

삼대각 시스템은 $n - 1$개의 미지수($c_1, \dots, c_{n-1}$)를 갖지만 내부 방정식은 $n - 3$개뿐이므로, **2개의 추가 경계(끝) 조건**이 필요하다:

| 끝 조건 | 설명 | 방정식 |
|---|---|---|
| **자연(Natural)** | 양 끝점에서 2차 도함수가 0 | $c_1 = 0, \quad c_n = 0$ |
| **고정(Clamped)** | 끝점에서의 1차 도함수 $f_1'$과 $f_n'$이 지정됨 | 삼대각 시스템의 첫 번째와 마지막 행을 수정 |
| **Not-a-knot** | 두 번째와 마지막에서 두 번째 매듭에서 **3차 도함수**의 연속성 | $S_1 = S_2$ (같은 삼차식)이고 $S_{n-2} = S_{n-1}$ (같은 삼차식) |

> **자연 스플라인**이 가장 일반적인 기본 선택이다. $S_1''(x_1) = 0$으로 설정하면 $c_1 = 0$을 얻고, $S_{n-1}''(x_n) = 0$으로 설정하면 ($(***)$를 통해) $c_n = 0$을 얻는다.
>
> 행렬 시스템의 크기를 $n \times n$으로 균일하게 만들기 위해, 마지막 구간이 $S_{n-1}$임에도 $c_n$을 추가 변수로 **도입하는 것이 편리**하다.

**자연 끝 조건의 유도:**

왼쪽 끝에서:

$$S_1''(x_1) = 2c_1 + 6d_1(x_1 - x_1) = 2c_1 = 0 \implies c_1 = 0$$

오른쪽 끝에서:

$$S_{n-1}''(x_n) = 2c_{n-1} + 6d_{n-1}h_{n-1} = 0$$

$c_{i+1} = c_i + 3d_i h_i$ (조건 $(***)$)를 사용하면, 이것은 $c_n = 0$을 의미한다.

### 5.5 예제 18.3 — 삼차 스플라인

**주어진 조건:** 4개의 데이터 점 $(x_1, f_1),\, (x_2, f_2),\, (x_3, f_3),\, (x_4, f_4)$

- 3개의 구간
- $3 \times 4 = 12$개의 미지수

**세 스플라인 함수 ($a_i = f_i$ 이후):**

$$S_1(x) = f_1 + b_1(x - x_1) + c_1(x - x_1)^2 + d_1(x - x_1)^3$$

$$S_2(x) = f_2 + b_2(x - x_2) + c_2(x - x_2)^2 + d_2(x - x_2)^3$$

$$S_3(x) = f_3 + b_3(x - x_3) + c_3(x - x_3)^2 + d_3(x - x_3)^3$$

**도함수:**

$$S_i'(x) = b_i + 2c_i(x - x_i) + 3d_i(x - x_i)^2$$

$$S_i''(x) = 2c_i + 6d_i(x - x_i)$$

**삼대각 시스템 (자연 끝 조건, $c_1 = 0$, $c_4 = 0$):**

$$\begin{pmatrix} 0 \\ 3(f[x_3, x_2] - f[x_2, x_1]) \\ 3(f[x_4, x_3] - f[x_3, x_2]) \\ 0 \end{pmatrix} = \begin{pmatrix} 1 & 0 & 0 & 0 \\ h_1 & 2(h_1 + h_2) & h_2 & 0 \\ 0 & h_2 & 2(h_2 + h_3) & h_3 \\ 0 & 0 & 0 & 1 \end{pmatrix} \begin{pmatrix} c_1 \\ c_2 \\ c_3 \\ c_4 \end{pmatrix}$$

**$c_1, c_2, c_3, c_4$를 풀고 나면, 복원:**

$$b_i = \frac{f_{i+1} - f_i}{h_i} - \frac{h_i}{3}(2c_i + c_{i+1}), \quad i = 1, 2, 3$$

$$d_i = \frac{c_{i+1} - c_i}{3h_i}, \quad i = 1, 2, 3$$

```python
import numpy as np

def cubic_spline_natural(x, f):
    """
    Compute natural cubic spline coefficients.

    Parameters
    ----------
    x : array-like, shape (n,)
        Knot positions (sorted).
    f : array-like, shape (n,)
        Function values at knots.

    Returns
    -------
    a, b, c, d : arrays, each of shape (n-1,)
        Coefficients for S_i(t) = a_i + b_i*(t-x_i)
                                    + c_i*(t-x_i)^2
                                    + d_i*(t-x_i)^3
    """
    n = len(x)
    h = np.diff(x)                     # h[i] = x[i+1] - x[i]

    # --- Build the tridiagonal system for c ---
    # Size: n x n  (with c[0] = 0, c[n-1] = 0 as natural BCs)
    A = np.zeros((n, n))
    rhs = np.zeros(n)

    # Natural end conditions
    A[0, 0] = 1.0
    A[n-1, n-1] = 1.0

    # Interior equations  (i = 1, 2, ..., n-2)
    for i in range(1, n - 1):
        A[i, i-1] = h[i-1]
        A[i, i]   = 2.0 * (h[i-1] + h[i])
        A[i, i+1] = h[i]
        rhs[i] = 3.0 * ((f[i+1] - f[i]) / h[i]
                       - (f[i] - f[i-1]) / h[i-1])

    c = np.linalg.solve(A, rhs)

    # --- Recover b and d ---
    a = f[:-1].copy()
    b = np.zeros(n - 1)
    d = np.zeros(n - 1)

    for i in range(n - 1):
        d[i] = (c[i+1] - c[i]) / (3.0 * h[i])
        b[i] = (f[i+1] - f[i]) / h[i] \
             - h[i] * (2.0 * c[i] + c[i+1]) / 3.0

    c = c[:-1]   # only need c[0] ... c[n-2]

    return a, b, c, d


def evaluate_spline(x_knots, a, b, c, d, x_eval):
    """Evaluate the cubic spline at points x_eval."""
    results = np.zeros_like(x_eval, dtype=float)
    for j, xv in enumerate(x_eval):
        # Find the correct interval
        i = np.searchsorted(x_knots, xv, side='right') - 1
        i = np.clip(i, 0, len(a) - 1)
        dx = xv - x_knots[i]
        results[j] = a[i] + b[i]*dx + c[i]*dx**2 + d[i]*dx**3
    return results


# --- Example usage ---
if __name__ == "__main__":
    # Example 18.3: 4 data points
    x = np.array([3.0, 4.5, 7.0, 9.0])
    f = np.array([2.5, 1.0, 2.5, 0.5])

    a, b, c, d = cubic_spline_natural(x, f)

    print("Spline coefficients:")
    for i in range(len(a)):
        print(f"  S_{i+1}(x): a={a[i]:.6f}, b={b[i]:.6f}, "
              f"c={c[i]:.6f}, d={d[i]:.6f}")

    # Evaluate
    x_test = np.array([5.0])
    y_test = evaluate_spline(x, a, b, c, d, x_test)
    print(f"\nS(5.0) = {y_test[0]:.6f}")
```

## 6. 쌍선형 보간(Bilinear Interpolation)

쌍선형 보간은 구간별 보간을 **2차원**으로 확장한다. 직사각형의 네 꼭짓점에서의 함수값이 주어지면, 임의의 내부 점 $(x_i, y_i)$에서의 값을 보간할 수 있다.

### 설정

꼭짓점이 $(x_1, y_1)$, $(x_2, y_1)$, $(x_1, y_2)$, $(x_2, y_2)$인 직사각형 셀을 고려한다. 각 꼭짓점에서의 함수값 $f(x_1, y_1)$, $f(x_2, y_1)$, $f(x_1, y_2)$, $f(x_2, y_2)$가 알려져 있다.

### 공식

$$f(x_i, y_i) = N_1(x_i, y_i)\,f(x_1, y_1) + N_2(x_i, y_i)\,f(x_2, y_2) + N_3(x_i, y_i)\,f(x_1, y_2) + N_4(x_i, y_i)\,f(x_2, y_1)$$

여기서 **쌍선형 기저 함수**는:

$$N_1(x, y) = \frac{x - x_2}{x_1 - x_2} \cdot \frac{y - y_2}{y_1 - y_2}$$

$$N_2(x, y) = \frac{x - x_1}{x_2 - x_1} \cdot \frac{y - y_2}{y_1 - y_2}$$

$$N_3(x, y) = \frac{x - x_2}{x_1 - x_2} \cdot \frac{y - y_1}{y_2 - y_1}$$

$$N_4(x, y) = \frac{x - x_1}{x_2 - x_1} \cdot \frac{y - y_1}{y_2 - y_1}$$

> **해석:** 각 $N_k$는 두 개의 1차원 선형 보간 가중치의 곱이다. $N_k$는 해당 꼭짓점에서 1이고 다른 모든 꼭짓점에서 0이다.

```python
def bilinear_interpolation(x1, x2, y1, y2, f11, f21, f12, f22, xi, yi):
    """
    Bilinear interpolation on a rectangular cell.

    f11 = f(x1, y1), f21 = f(x2, y1)
    f12 = f(x1, y2), f22 = f(x2, y2)
    """
    N1 = ((x2 - xi) / (x2 - x1)) * ((y2 - yi) / (y2 - y1))   # corner (x1,y1)
    N2 = ((xi - x1) / (x2 - x1)) * ((y2 - yi) / (y2 - y1))   # corner (x2,y1)
    N3 = ((x2 - xi) / (x2 - x1)) * ((yi - y1) / (y2 - y1))   # corner (x1,y2)
    N4 = ((xi - x1) / (x2 - x1)) * ((yi - y1) / (y2 - y1))   # corner (x2,y2)

    return N1 * f11 + N2 * f21 + N3 * f12 + N4 * f22
```

## 7. 요약 표

| 스플라인 유형 | 다항식 차수 | 연속성 | 구간당 미지수 | 필요한 추가 조건 | 핵심 방정식 |
|---|---|---|---|---|---|
| **선형** | 1 | $C^0$ | 2 ($a_i, b_i$) | 없음 | $S_i(x) = f_i + \frac{f_{i+1}-f_i}{h_i}(x-x_i)$ |
| **이차** | 2 | $C^1$ | 3 ($a_i, b_i, c_i$) | 1 ($c_1=0$ 설정) | $S_i(x) = a_i + b_i(x-x_i) + c_i(x-x_i)^2$ |
| **삼차** | 3 | $C^2$ | 4 ($a_i, b_i, c_i, d_i$) | 2 (끝 조건) | $S_i(x) = a_i + b_i(x-x_i) + c_i(x-x_i)^2 + d_i(x-x_i)^3$ |

| 성질 | 선형 | 이차 | 삼차 |

| 계산 비용 | $O(n)$ | $O(n)$ | $O(n)$ (토머스 알고리즘 사용) |
| 대표적 응용 | 단순 보간 | 중간 수준의 매끄러움 | 컴퓨터 그래픽스, 공학 |
